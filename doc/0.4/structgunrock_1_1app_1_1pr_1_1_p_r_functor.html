Detailed Description

template<typename VertexId, typename SizeT, typename Value, typename Problem>
struct gunrock::app::pr::PRFunctor< VertexId, SizeT, Value, Problem >

Structure contains device functions in PR graph traverse.

Template Parameters
VertexIdType of signed integer to use as vertex identifier.
SizeTType of unsigned integer to use for array indexing.
ValueType of float or double to use for computed values.
ProblemProblem data type which contains data slice for problem.
